# 完整的URL


- 协议
    - 双方
        - 客户端: 浏览器（发送请求的软件）
        - 服务端: web server（接收请求的软件）
    - 内容
        - 定义接收和发送的方式
- 域名
    - IP的别名
    - 完整的域名分3部分 前缀 名 后缀
    - 一般所说的域名是 名 后缀
- 接口
- 参数
- 文件路径
